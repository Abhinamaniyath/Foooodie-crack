# Foooodie-crack

A modern responsive restaurant website designed to showcase delicious dishes, talented chefs, and customer experiences .This website provides a smooth and attractive browsing experience.



##  Features
- Responsive design for all devices  
- Attractive food gallery  
- Menu and chef sections  
- Customer testimonials  
- Contact form with Google Map

- Bootstrap 5  
- JavaScript
